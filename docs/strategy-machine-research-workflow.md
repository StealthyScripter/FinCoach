# Strategy Machine Research Workflow

MarketPilot now centers on a demo-only scientific strategy loop for forex, metals, and selected individual stock fixtures:

1. Observe normalized market snapshots and candles.
2. Detect deterministic patterns.
3. Generate evidence-linked hypotheses.
4. Build objective rule sets.
5. Create experiments as the learning unit.
6. Backtest with spread, slippage, commission, and R-multiple metrics.
7. Validate statistically before forward testing.
8. Forward test only in demo-like modes.
9. Journal every trade and snapshot.
10. Rank, promote, demote, retire, or refine based on evidence.

The workflow preserves event lineage with correlation IDs, causation IDs, source event references, schema versions, and immutable event envelopes.
