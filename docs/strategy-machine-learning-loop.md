# Strategy Machine Continuous Improvement Loop

The learning loop is:

Observe -> Detect Pattern -> Generate Hypothesis -> Define Rules -> Backtest -> Validate -> Forward Test -> Journal -> Review -> Refine -> Retest.

Refinement never mutates old strategy evidence. It emits a refinement suggestion, creates a new experiment version, creates a new rule-set version reference, requests retest, and completes the loop with lineage back to journal evidence.
